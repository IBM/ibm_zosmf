# Ansible role: zmf_swmgmt_holds_for_reason
The collection [ibm_zosmf](../../README.md) provides an [Ansible role](https://docs.ansible.com/ansible/latest/user_guide/playbooks_reuse_roles.html), referred to as `zmf_swmgmt_holds_for_reason`, to separate HOLDs by REASON and run the csi query REST API to get the HOLDDATA text.

For guides and reference, see [Docs Site](https://ibm.github.io/z_ansible_collections_doc/ibm_zosmf/docs/source/roles/zmf_swmgmt_holds_for_reason.html).
